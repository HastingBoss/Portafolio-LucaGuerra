# Portafolio - Luca Guerra

A modern and responsive professional portfolio showcasing my work as a Full Stack Web Development student specialized in Frontend. The project is structured around clean UI aesthetics, performance, and responsive design.

## Built With

* React
* Vite
* Framer Motion
* CSS (Custom properties, minimal glassmorphism layout)

## Features

* Single Page Application architecture
* Fully responsive layout handling mobile displays down to 320px
* Interactive UI transitions and hover effects
* Custom Image Carousel for project showcases
* Optimized SEO and Open Graph metadata
